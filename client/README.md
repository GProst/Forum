# Forum client

## Install

* Run:

```shell
npm install
```

## Start dev server

* Run

```shell
npm start -- --output-public-path "/" --port 3000
```

## Build

* Development version:

```shell
npm run build-dev -- --output-public-path "/"
```

* Production version:

```shell
npm run build-prod -- --output-public-path "/"
```

___

## Used technologies

* ReactJS

* Redux

* Webpack

* Babel

* Styled-components

* Eslint
