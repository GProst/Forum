# Simple forum client

## Install

* Run:

```shell
npm install
```

## Start dev server

* Run

```shell
npm start -- --output-public-path "http://localhost:3000/" --port 3000
```

## Build

* Development version:

```shell
npm run build-dev -- --output-public-path "http://localhost:3000/"
```

* Production version:

```shell
npm run build-prod -- --output-public-path "http://localhost:3000/"
```

___
