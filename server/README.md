# Forum server

## Install

* Run:

```shell
npm install
```

## Initialize DB

* Run:

```shell
npm run initialize-db
```

## Bootstrap server

> **NOTE:** initialize DB before start server!

* For development:

```shell
npm run start-dev --port 3000
```

* For production:

```shell
npm run start --port 3000
```

## Test

```shell
npm test
```

___

## Used technologies

* Express.js

* SQLite 3
